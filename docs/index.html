turtle-matter generates vocabulary documentation from RDF files. Below are example outputs showing different formats generated from the same source vocabulary.

The HTML pages contain machine-readable JSON-LD information embedded in the document header, while the Markdown files include the JSON-LD context in their frontmatter.

When companion documentation in per-term Markdown files are provided, they are included in the generated outputs for each term.

Examples

Croissant Vocabulary

MLCommons Croissant vocabulary for ML dataset metadata

Inputs

  • croissant.ttl Turtle
  • Documentation files Markdown
    • DataSource (raw)
    • DataType (raw)
    • Extract (raw)
    • Field (raw)
    • FileObject (raw)
    • FileSet (raw)
    • Format (raw)
    • RecordSet (raw)
    • Transform (raw)

Outputs

  • Interactive HTML Documentation HTML
  • Markdown Documentation (raw) Markdown
  • JSON-LD Context JSON-LD
